# Design Brief: API Spending Widget (Chrome Extension)

## Product

Chrome-расширение, которое в реальном времени показывает расходы
на AI API (Anthropic/Claude, OpenAI, Google Gemini). Попап открывается
по клику на иконку в тулбаре Chrome.

## Маскот

**Claude Toad** — пиксельная зелёная жаба. Уже существует, анимирована
(трекинг глаз за курсором, моргание, квакание по клику, speech bubbles).
Ассеты приложены в папке `design/`:
- `claude-toad.png` — основной арт
- `claude-toad-clear-no-eyes.png` — тело без глаз (для анимации зрачков)
- `claude-toad-clear-eyes-only.png` — только глаза (2 КБ, для оверлея)
- `claude-toad-square.png` — квадратная версия (для иконки расширения)
- `claude-toad-body.png` — тело для компонента

---

## Что нужно спроектировать

### 1. Popup (основной виджет, 370px шириной)

Содержание (зафиксировано, менять нельзя):
- Заголовок с жабой и названием "API Spending"
- "updated X min ago" — метка свежести данных
- Три карточки: Today / This Week / This Month
  - Today выделена, показывает сумму + конвертацию (напр. $4.23 / £3.35)
- Раздел "By Provider" — Anthropic, OpenAI, Gemini
  - Каждый с прогресс-баром и суммой
- Раздел "Recent Requests" — последние 4-5 запросов
  - Время, цветная точка провайдера, модель, стоимость
- Footer: кнопки Settings и Refresh

Что решает дизайнер:
- Визуальный стиль, типографика, цветовая палитра
- Компоновка и spacing
- Как интегрировать жабу (маленькая иконка? анимированный элемент? фоновый паттерн?)
- Стиль карточек, баров, списка
- Микроанимации и hover-эффекты
- Тёмная тема обязательна, но палитра на усмотрение дизайнера

### 2. Mini badge (иконка в тулбаре Chrome, 16-28px)

Это то, что пользователь видит ВСЕГДА — маленькая иконка расширения
в панели Chrome рядом с адресной строкой.

- Квадратная иконка жабы (claude-toad-square.png как база)
- Бейдж с суммой за сегодня (напр. "$4") — зелёный или в стиле палитры
- Должно быть читаемо при 16x16

### 3. Settings page (опционально, бонус)

- Ввод Admin API ключей по провайдерам
- Выбор основной валюты (dropdown)
- Переключатели провайдеров (вкл/выкл)
- Budget alert — порог уведомления

## Стилистические ориентиры

Дизайнер выбирает направление сам, но вот возможные вектора:

**Вариант A — "Claude style"**
Вдохновлён интерфейсом claude.ai: тёплые кремовые тона на тёмном фоне,
мягкие скругления, шрифт с характером. Минимализм, много воздуха.

**Вариант B — "Terminal / hacker"**
Моноширинный шрифт, зелёное на чёрном, CLI-эстетика.
Жаба как ASCII-арт или пиксельная вставка.

**Вариант C — "Dashboard"**
Инфографика, sparklines, маленькие графики трендов.
Более "продуктовый" вид, ближе к Stripe Dashboard.

**Вариант D — что-то своё**
Дизайнер может предложить собственное направление.

Пожелания:
- Не перегружать. Это маленький виджет, не SaaS-приложение.
- Жаба — изюминка, а не помеха. Она должна быть заметна,
  но не доминировать над данными.
- Числа — главный элемент. Сумма за день должна
  считываться за 0.5 секунды.
- Моноширинный шрифт для цифр обязателен.

## Ограничения

- Ширина попапа: 370px (фиксированная для Chrome extension)
- Высота: ~500-600px максимум (скролл нежелателен)
- Тёмная тема (светлую делать не надо)
- Нет внешних изображений кроме жабы (иконки — SVG inline)
- Формат deliverables: Figma или HTML-макет

---

## Deliverables

1. Popup — 2-3 варианта стилистики (хотя бы скетч-уровень)
2. Mini badge — как выглядит иконка в тулбаре Chrome
3. Один вариант — до финального качества с hover-состояниями
4. Цветовая палитра и типографика (токены)
